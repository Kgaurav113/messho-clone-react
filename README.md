### Steps to deploy
```
  git clone https://github.com/ // Clone This Repo
  cd Meesho                                                // Move to the repo
  npm install                                              // install required dependencies
  npm run start                                            // Run all Servers
```

Now Visit  [localhost:3000](http://localhost:3000)
